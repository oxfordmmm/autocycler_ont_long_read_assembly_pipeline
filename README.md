# autocycler_ont_long_read_assembly_pipeline
A Nextflow pipeline for generating reference-grade consensus assemblies of clinical Enterobacterales using Oxford Nanopore Technologies (ONT) long-read DNA sequences and the consensus assembler Autocycler, with downstream quality control (CheckM2, seqkit stats) and annotation (MLST, Kraken2, Bakta, AMRFinder PLUS, MOB-suite).
